# Financial System Program — Improvement Backlog

Prioritized, cross-sprint upgrades for the compliance-first financial system.

## P0 (Critical)

1. **Consent Validation Instrument**
   - Verify each contact has legal basis for outreach.
   - Block campaign export when consent status is missing.

2. **Contact Dedup + Suppression Engine**
   - Merge duplicate identities across inbox/sent/CC sources.
   - Enforce suppression lists (unsubscribed, bounced, blocked).

3. **Transfer Ledger Automation**
   - Log transfer date, amount, destination, and confirmation ID.
   - Reconcile ledger against weekly revenue totals.

## P1 (High)

1. **Orange Segmentation Pipeline**
   - Automate periodic exports for engagement and relevance clusters.
   - Feed segment summaries back into RAG knowledge.

2. **Funnel Experiment Harness**
   - Track lead magnet performance by traffic source.
   - Auto-rank winning copy/offer variants by conversion.

3. **KPI Dashboard**
   - MRR, CAC proxy, opt-in conversion, net margin, and content cadence.

## P2 (Medium)

1. **Trading Research Sandbox**
   - Backtest + paper-trade gate before any live deployment.
   - Log risk metrics (max drawdown, sharpe proxy, win/loss streak).

2. **Story Engine Templates**
   - Script template for tutorial episodes.
   - Character style guide for anthropomorphic narrator.

3. **Weekly Retrospective Pack**
   - Auto-generate wins, blockers, and next experiments.
