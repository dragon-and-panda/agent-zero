## Launch Order

1. Sell a narrowly scoped **Inbox-to-CRM Hygiene** offer to prove fast delivery and cash conversion.
2. Run the **Autonomous Listing Concierge** in parallel as a second lane for redundancy.
3. Use spare capacity for **Public-Web Research Briefs** to smooth demand swings.
4. Convert documented wins into the **Course + Character Content Funnel** once case studies exist.

## Required Notes Per New Opportunity

When adding an item, capture:

- target customer,
- revenue model,
- data sources,
- consent status,
- time to first dollar,
- automation leverage,
- failure modes,
- fallback plan if acquisition stalls.
