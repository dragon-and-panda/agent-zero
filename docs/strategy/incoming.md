# Strategy Intake Queue

Use this queue for new venture ideas before resources are committed. Score each item with `instruments/strategy/score.sh` and record the resulting decision.

## Intake Fields
- Opportunity
- Customer
- Data source
- Why now
- Delivery mode
- Legality (1-10)
- Time to cash (1-10)
- Automation fit (1-10)
- Margin potential (1-10)
- Repeatability (1-10)
- Downside safety (1-10)
- Notes
